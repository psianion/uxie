# Uxie Implementation Plan

> **Plan v1.1 (2026-04-27)** — aligned with design v1.1 and the spec layer (`docs/SUP-GUIDELINES.md`, `docs/UXIE-DISCORD-GUIDELINES.md`, `docs/discordjs-api-surface.md`). See revision log at the bottom of this file.

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build uxie v1 — a single-user Discord bot with one active module (`scrypt`) exposing `/capture`, `/ask`, `/search`, `/journal`, `/brief`, `/ping`, and a passive `#inbox` channel capture, running on Docker Desktop for development.

**Architecture:** One long-lived Bun process running discord.js v14.26. Outward-facing module folder under `src/integrations/scrypt/` holds the REST + MCP clients and the command files. A sibling `src/integrations/para-raid/` folder holds the v2 module placeholder (`orchestrator-stub.ts`). Writes go through `POST /api/ingest` (scrypt owns path/slug). Reads use MCP streamable-http (`searchNotes`, `semanticSearch`, `getNote`). No local state, no queue, no scheduler, owner-gated, error-boundary per interaction.

**Tech Stack:** Bun 1.x, TypeScript, discord.js `^14.26`, zod, `bun test`, Docker Desktop. No Node toolchain, no npm — `bun` all the way.

**Spec:** `docs/superpowers/specs/2026-04-14-uxie-design.md` (v1.1)

**Execution model:** Sequential — tasks have tight ordering dependencies (shared `lib/` grows across waves, rest-client grows across waves). One task at a time. Commit after each task.

```ts
// src/lib/env.ts
// SOLE source of process.env access — no other module reads env directly (UXIE-DISCORD-GUIDELINES §17).
import { z } from "zod";
import { ConfigError } from "./errors.ts";

const schema = z.object({
  DISCORD_BOT_TOKEN: z.string().min(1),
  DISCORD_APP_ID: z.string().min(1),
  DISCORD_DEV_GUILD_ID: z.string().min(1),
  DISCORD_OWNER_ID: z.string().min(1),
  INBOX_CHANNEL_ID: z.string().min(1),
  USER_TZ: z.string().min(1),
  SCRYPT_SERVER_URL: z.string().url(),
  SCRYPT_MCP_URL: z.string().url(),
  SCRYPT_AUTH: z.string().min(1),
});

export type Env = z.infer<typeof schema>;

export function parseEnv(src: Record<string, string | undefined> = process.env): Env {
  const result = schema.safeParse(src);
  if (!result.success) {
    // Surface the failed field name(s) so the boot log + the operator know exactly what to fix.
    const fields = result.error.issues.map((i) => i.path.join(".")).join(", ");
    throw new ConfigError("config", `invalid env (missing/invalid: ${fields})`, result.error);
  }
  return result.data;
}
```

Logger redaction (Task 4) is the partner here: any `Fields` object containing keys with `BOT_TOKEN`/`AUTH`/`SECRET`/`KEY` substrings will be redacted before serialisation, so accidentally logging the parsed env never leaks secrets.

```ts
// src/lib/log.ts
type Level = "info" | "warn" | "error";
type Fields = Record<string, unknown>;

export interface Logger {
  info(msg: string, fields?: Fields): void;
  warn(msg: string, fields?: Fields): void;
  error(msg: string, fields?: Fields): void;
  child(scope: Fields): Logger;
}

// UXIE-DISCORD-GUIDELINES §17.2 — never let secrets reach stdout.
const REDACT_SUBSTRINGS = ["BOT_TOKEN", "AUTH", "SECRET", "KEY"] as const;
function shouldRedact(key: string): boolean {
  const upper = key.toUpperCase();
  return REDACT_SUBSTRINGS.some((s) => upper.includes(s));
}

function serializeField(v: unknown): unknown {
  if (v instanceof Error) return `${v.name}: ${v.message}\n${v.stack ?? ""}`;
  return v;
}

function emit(level: Level, scope: Fields, msg: string, fields: Fields = {}) {
  const merged: Fields = { t: new Date().toISOString(), level, msg };
  for (const [k, v] of Object.entries(scope)) merged[k] = shouldRedact(k) ? "[REDACTED]" : serializeField(v);
  for (const [k, v] of Object.entries(fields)) merged[k] = shouldRedact(k) ? "[REDACTED]" : serializeField(v);
  console.log(JSON.stringify(merged));
}

function make(scope: Fields): Logger {
  return {
    info: (msg, f) => emit("info", scope, msg, f),
    warn: (msg, f) => emit("warn", scope, msg, f),
    error: (msg, f) => emit("error", scope, msg, f),
    child: (extra) => make({ ...scope, ...extra }),
  };
}

export function createLogger(): Logger { return make({}); }
export const log = createLogger();
```

```ts
// src/lib/tz.ts
// USER_TZ helpers used by /journal reply text and /brief title.
// Render-only: scrypt owns storage-side TZ math; these helpers never decide file paths.

export function journalDateKey(tz: string, date: Date = new Date()): string {
  const fmt = new Intl.DateTimeFormat("en-CA", {
    timeZone: tz,
    year: "numeric",
    month: "2-digit",
    day: "2-digit",
  });
  // en-CA gives YYYY-MM-DD natively.
  return fmt.format(date);
}

export function nowInZone(tz: string, date: Date = new Date()): { date: string; time: string } {
  const dateStr = journalDateKey(tz, date);
  const timeFmt = new Intl.DateTimeFormat("en-GB", {
    timeZone: tz,
    hour: "2-digit",
    minute: "2-digit",
    hour12: false,
  });
  return { date: dateStr, time: timeFmt.format(date) };
}
```

```ts
import {
  ApplicationIntegrationType,
  InteractionContextType,
  type SlashCommandBuilder,
} from "discord.js";

/**
 * UXIE-DISCORD-GUIDELINES §6.2 — every slash-command builder must apply these three setters.
 * Centralised here so the six command files cannot drift.

export function withOwnerGate<B extends SlashCommandBuilder>(b: B): B {
  b.setContexts(InteractionContextType.Guild);
  b.setIntegrationTypes(ApplicationIntegrationType.GuildInstall);
  b.setDefaultMemberPermissions(0n);
  return b;
}
```

```md
Single-user Discord bot hosting outward-facing modules (scrypt in v1).

## Discord prerequisites

1. Create an application at https://discord.com/developers/applications
2. Create a bot user; copy the token → `.env` `DISCORD_BOT_TOKEN`
3. Copy the application id → `DISCORD_APP_ID`
```

```ts
// src/integrations/scrypt/rest-client.ts
import { ScryptError } from "../../lib/errors.ts";

export interface HealthResult { ok: boolean; reason?: "unreachable" | "auth" | "server" | "timeout"; }

export class ScryptRestClient {
  constructor(private baseUrl: string, private bearer: string) {}

  private headers(): Record<string, string> {
    return { "Authorization": `Bearer ${this.bearer}`, "Content-Type": "application/json" };
  }

  async health(): Promise<HealthResult> {
    try {
      const res = await fetch(`${this.baseUrl}/api/daily_context`, {
        method: "GET",
        headers: this.headers(),
        signal: AbortSignal.timeout(500),
      });
      if (res.status === 200) return { ok: true };
      if (res.status === 401 || res.status === 403) return { ok: false, reason: "auth" };
      if (res.status >= 500) return { ok: false, reason: "server" };
      return { ok: false, reason: "server" };
    } catch (e) {
      if (e instanceof DOMException && e.name === "TimeoutError") return { ok: false, reason: "timeout" };
      return { ok: false, reason: "unreachable" };
    }
  }
}
```

```ts
// src/bot/command-loader.ts
import { Collection, type SlashCommandBuilder, type ChatInputCommandInteraction } from "discord.js";

export interface CommandContext {
  clientTag: string;
  log: { info: (m: string, f?: Record<string, unknown>) => void; warn: (m: string, f?: Record<string, unknown>) => void; error: (m: string, f?: Record<string, unknown>) => void; };
}

export interface LoadedCommand {
  data: SlashCommandBuilder | { name: string };
  execute: (i: ChatInputCommandInteraction, ctx: CommandContext) => Promise<void>;
}

export function buildCommandCollection(cmds: LoadedCommand[]): Collection<string, LoadedCommand> {
  const c = new Collection<string, LoadedCommand>();
  for (const cmd of cmds) {
    const name = cmd.data.name;
    if (c.has(name)) throw new Error(`duplicate command name: ${name}`);
    c.set(name, cmd);
  }
  return c;
}
```

```ts
// src/bot/interaction-router.ts
import {
  MessageFlags,
  type ChatInputCommandInteraction,
  type Collection,
  type Interaction,
} from "discord.js";
import { UxieError, NotOwnerError, ScryptError } from "../lib/errors.ts";
import { assertOwner } from "../lib/auth.ts";
import { makeClientTag } from "../lib/client-tag.ts";
import { log } from "../lib/log.ts";
import type { LoadedCommand } from "./command-loader.ts";

export async function handleInteraction(
  i: Interaction,
  commands: Collection<string, LoadedCommand>,
  ownerId: string,
): Promise<void> {
  if (!i.isChatInputCommand()) return;
  const cmd = commands.get(i.commandName);
  if (!cmd) return;

  const scoped = log.child({ interactionId: i.id, command: i.commandName });
  try {
    assertOwner(i, ownerId);
    await (i as ChatInputCommandInteraction).deferReply({ flags: MessageFlags.Ephemeral });
    await cmd.execute(i as ChatInputCommandInteraction, { clientTag: makeClientTag(i), log: scoped });
    scoped.info("command ok");
  } catch (err) {
    await replyWithError(i as ChatInputCommandInteraction, err, scoped);
  }
}

async function replyWithError(
  i: ChatInputCommandInteraction,
  err: unknown,
  scoped: { info: Function; warn: Function; error: Function },
) {
  if (err instanceof NotOwnerError) {
    scoped.info("not owner");
    if (!i.deferred && !i.replied) await i.reply({ content: "not for you", flags: MessageFlags.Ephemeral });
    return;
  }
  if (err instanceof ScryptError) {
    scoped.warn("scrypt error", { err });
    const msg = err.message;
    if (i.deferred) await i.editReply(msg);
    else await i.reply({ content: msg, flags: MessageFlags.Ephemeral });
    return;
  }
  if (err instanceof UxieError) {
    scoped.warn("uxie error", { err });
    if (i.deferred) await i.editReply(`uxie: ${err.message}`);
    else await i.reply({ content: `uxie: ${err.message}`, flags: MessageFlags.Ephemeral });
    return;
  }
  scoped.error("unhandled", { err });
  if (i.deferred) await i.editReply("uxie crashed, check logs");
  else await i.reply({ content: "uxie crashed, check logs", flags: MessageFlags.Ephemeral });
}
```

```ts
export interface IngestParams {
  kind: string;
  content: string;
  clientTag: string;
  tz?: string;
  meta?: Record<string, unknown>;
}

export interface IngestResult { path: string; permalink: string; }

// inside the class:
async ingest(p: IngestParams): Promise<IngestResult> {
  const body: Record<string, unknown> = {
    kind: p.kind,
    content: p.content,
    client_tag: p.clientTag,
  };
  if (p.tz !== undefined) body.tz = p.tz;
  if (p.meta !== undefined) body.meta = p.meta;

  let res: Response;
  try {
    res = await fetch(`${this.baseUrl}/api/ingest`, {
      method: "POST",
      headers: this.headers(),
      body: JSON.stringify(body),
      signal: AbortSignal.timeout(10_000),
    });
  } catch (e) {
    const name = (e as Error).name;
    if (name === "TimeoutError") throw new ScryptError("scrypt_timeout", "scrypt timed out", e);
    throw new ScryptError("scrypt_unreachable", "scrypt unreachable", e);
  }

  if (res.status === 200) {
    const parsed = (await res.json()) as IngestResult;
    return parsed;
  }
  if (res.status === 401 || res.status === 403) throw new ScryptError("scrypt_auth", "scrypt auth rejected");
  if (res.status >= 500) throw new ScryptError("scrypt_server", "scrypt server error");
  const msg = await res.text().catch(() => "");
  throw new ScryptError("scrypt_bad_request", `scrypt: ${msg || res.statusText}`);
}
```

```ts
// src/bot/message-router.ts
import type { Message } from "discord.js";
import { isInboxChannel } from "../integrations/scrypt/channels.ts";

export interface MessageRouterConfig { ownerId: string; inboxId: string; }

export async function handleMessage(
  msg: Message,
  cfg: MessageRouterConfig,
  onInbox: (msg: Message) => Promise<void>,
): Promise<void> {
  if (msg.author.bot) return;
  if (msg.author.id !== cfg.ownerId) return;
  if (!isInboxChannel(msg.channelId, cfg.inboxId)) return;
  if (!msg.content || msg.content.trim() === "") return;
  await onInbox(msg);
}
```

```ts
// src/integrations/scrypt/mcp-client.ts
import { ScryptError } from "../../lib/errors.ts";

export interface SearchHit { note_path: string; match_preview: string; }
export interface SemanticHit { note_path: string; chunk_text: string; score: number; }
export interface Note { path: string; title: string; body: string; }

export class ScryptMcpClient {
  constructor(private url: string, private bearer: string) {}

  async searchNotes(query: string, limit: number): Promise<SearchHit[]> {
    return this.call<SearchHit[]>("search_notes", { query, limit });
  }

  async semanticSearch(query: string, limit: number): Promise<SemanticHit[]> {
    return this.call<SemanticHit[]>("semantic_search", { query, limit });
  }

  async getNote(path: string): Promise<Note> {
    return this.call<Note>("get_note", { path });
  }

  private async call<T>(tool: string, args: Record<string, unknown>): Promise<T> {
    const body = { jsonrpc: "2.0", id: 1, method: "tools/call", params: { name: tool, arguments: args } };
    let res: Response;
    try {
      res = await fetch(this.url, {
        method: "POST",
        headers: {
          "Authorization": `Bearer ${this.bearer}`,
          "Content-Type": "application/json",
          "Accept": "application/json",
        },
        body: JSON.stringify(body),
        signal: AbortSignal.timeout(10_000),
      });
    } catch (e) {
      const name = (e as Error).name;
      if (name === "TimeoutError") throw new ScryptError("scrypt_timeout", "scrypt timed out", e);
      throw new ScryptError("scrypt_unreachable", "scrypt unreachable", e);
    }
    if (res.status === 401 || res.status === 403) throw new ScryptError("scrypt_auth", "scrypt auth rejected");
    if (res.status >= 500) throw new ScryptError("scrypt_server", "scrypt server error");
    if (res.status !== 200) {
      const txt = await res.text().catch(() => "");
      throw new ScryptError("scrypt_bad_request", `scrypt: ${txt || res.statusText}`);
    }
    const parsed = (await res.json()) as { content?: Array<{ type: string; text: string }>; isError?: boolean; };
    if (parsed.isError) throw new ScryptError("scrypt_tool_error", "scrypt tool failed");
    const textPart = parsed.content?.find((p) => p.type === "text")?.text;
    if (!textPart) throw new ScryptError("scrypt_tool_error", "scrypt returned empty tool result");
    return JSON.parse(textPart) as T;
  }
}
```

```ts
import type { SearchHit, SemanticHit } from "../integrations/scrypt/mcp-client.ts";

export function searchResultEmbed(query: string, hits: SearchHit[]): EmbedBuilder {
  if (hits.length === 0) {
    return new EmbedBuilder().setTitle(`search: ${truncate(query, 100)}`).setDescription("no matches");
  }
  const lines = hits.map((h) => `• \`${h.note_path}\` — ${truncate(h.match_preview, 180)}`);
  return new EmbedBuilder()
    .setTitle(`search: ${truncate(query, 100)}`)
    .setDescription(truncate(lines.join("\n"), DESC_CAP));
}

export function semanticResultEmbed(query: string, hits: SemanticHit[]): EmbedBuilder {
  if (hits.length === 0) {
    return new EmbedBuilder().setTitle(`ask: ${truncate(query, 100)}`).setDescription("no matches");
  }
  const lines = hits.map((h) => `• \`${h.note_path}\` (${h.score.toFixed(2)}) — ${truncate(h.chunk_text, 200)}`);
  return new EmbedBuilder()
    .setTitle(`ask: ${truncate(query, 100)}`)
    .setDescription(truncate(lines.join("\n"), DESC_CAP));
```

```ts
// append to src/lib/embed.ts
import type { DailyContext } from "../integrations/scrypt/rest-client.ts";

export function briefEmbed(ctx: DailyContext, date: string): EmbedBuilder {
  const journalVal = ctx.today_journal ? truncate(ctx.today_journal, 400) : "_(empty)_";
  const threadsVal = ctx.open_threads.length === 0
    ? "_(none)_"
    : truncate(ctx.open_threads.slice(0, 5).map((t) => `• [${t.priority}] \`${t.path}\``).join("\n"), FIELD_CAP);
  const recentVal = ctx.recent_notes.length === 0
    ? "_(none)_"
    : truncate(ctx.recent_notes.slice(0, 5).map((n) => `• \`${n.path}\``).join("\n"), FIELD_CAP);
  const memVal = ctx.active_memories.length === 0
    ? "_(none)_"
    : truncate(ctx.active_memories.map((m) => m.name).join(", "), FIELD_CAP);
  const tagsVal = ctx.tag_cloud.length === 0
    ? "_(none)_"
    : truncate(ctx.tag_cloud.slice(0, 10).map((t) => `\`${t.tag}\`(${t.count})`).join(" "), FIELD_CAP);

  return new EmbedBuilder()
    .setTitle(`Daily brief — ${date}`)
    .addFields(
      { name: "Today's journal", value: journalVal },
      { name: "Open threads", value: threadsVal },
      { name: "Recent captures", value: recentVal },
      { name: "Active memories", value: memVal },
      { name: "Tags", value: tagsVal },
    );
}
```

```ts
// No try/catch in command body — interaction-router is the only catch site (UXIE §14.2 + §22).
import { SlashCommandBuilder } from "discord.js";
import type { LoadedCommand } from "../../../bot/command-loader.ts";
import type { ScryptRestClient } from "../rest-client.ts";
import { briefEmbed } from "../../../lib/embed.ts";
import { withOwnerGate } from "../../../lib/command-builder.ts";
import { journalDateKey } from "../../../lib/tz.ts";

export function buildBriefCommand(rest: ScryptRestClient, userTz: string): LoadedCommand {
  return {
    data: withOwnerGate(
      new SlashCommandBuilder().setName("brief").setDescription("Today's scrypt daily brief"),
    ),
    async execute(i) {
      const ctx = await rest.getDailyContext();
      const date = journalDateKey(userTz);
      await i.editReply({ embeds: [briefEmbed(ctx, date)] });
    },
  };
}
```

```ts
import { buildCommandCollection, type LoadedCommand } from "../../bot/command-loader.ts";
import { ScryptRestClient } from "./rest-client.ts";
import { ScryptMcpClient } from "./mcp-client.ts";
import { buildPingCommand } from "./commands/ping.ts";
import { buildCaptureCommand } from "./commands/capture.ts";
import { buildSearchCommand } from "./commands/search.ts";
import { buildAskCommand } from "./commands/ask.ts";
import { buildJournalCommand } from "./commands/journal.ts";

import { handleInboxMessage } from "./inbox-handler.ts";
import type { Env } from "../../lib/env.ts";
import type { Message } from "discord.js";
import { log } from "../../lib/log.ts";

export interface ScryptModule {
  commands: ReturnType<typeof buildCommandCollection>;
  rest: ScryptRestClient;
  mcp: ScryptMcpClient;
  onInbox: (msg: Message) => Promise<void>;
}

export function buildScryptModule(env: Env): ScryptModule {
  const rest = new ScryptRestClient(env.SCRYPT_SERVER_URL, env.SCRYPT_AUTH);
  const mcp = new ScryptMcpClient(env.SCRYPT_MCP_URL, env.SCRYPT_AUTH);
  const cmds: LoadedCommand[] = [
    buildPingCommand(rest),
    buildCaptureCommand(rest),
    buildSearchCommand(mcp),
    buildAskCommand(mcp),
    buildJournalCommand(rest, env.USER_TZ),
    buildBriefCommand(rest, env.USER_TZ),
  ];
  return {
    commands: buildCommandCollection(cmds),
    rest,
    mcp,
    onInbox: (msg) => handleInboxMessage(msg, rest, log),
  };
}
```
